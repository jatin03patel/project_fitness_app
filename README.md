## ✨ Features

- 🔍 Search exercises by name, body part, or equipment
- 📽 View exercise GIFs and video links
- 📱 Fully responsive design (mobile + desktop)
- 🧠 Clean and modern UI/UX
- 💡 Fetched from RapidAPI's ExerciseDB

---

## 🛠 Tech Stack

- ⚛️ React.js
- 🎨 Material UI
- 🌐 RapidAPI (ExerciseDB)
- 📦 React Router
